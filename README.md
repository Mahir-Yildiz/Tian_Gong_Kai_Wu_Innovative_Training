# Tian_Gong_Kai_Wu_Innovative_Training
这是一款融入可拓创新思维方法的小游戏，寓教于乐。既可初步了解可拓学，又能认识《天工开物》中各类器物的独特价值。希望大家玩得开心哦~ This is an educational game that incorporates Extenics innovation methodology, combining learning with entertainment. It allows players to gain a basic understanding of Extenics while exploring the unique value of various artifacts from Tiangong Kaiwu. Hope you all enjoy it!
